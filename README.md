## Sobre o Projeto

O **Maestro4Noobs** é um guia introdutório para quem quer aprender automação de testes mobile usando o [Maestro](https://maestro.mobile.dev/), um framework moderno criado pela Block (ex-Square) para testar aplicações iOS e Android com sintaxe simples em YAML.

Ideal para quem está começando em QA mobile ou quer uma alternativa mais simples ao Appium.

### Pré-requisitos

- Conhecimento básico de linha de comando (terminal)
- Android Studio instalado (para emulador Android) ou Xcode (para iOS)
- Curiosidade e vontade de aprender!
